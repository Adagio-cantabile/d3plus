## Installing

If you use NPM, `npm install d3plus@next`. Otherwise, download the [latest release](https://github.com/alexandersimoes/d3plus/releases/latest). The released bundle supports AMD, CommonJS, and vanilla environments. You can also load directly from [d3plus.org](https://d3plus.org):

```html
<script src="https://d3plus.org/js/d3plus.v2.0.0-alpha.13.full.min.js"></script>
```
